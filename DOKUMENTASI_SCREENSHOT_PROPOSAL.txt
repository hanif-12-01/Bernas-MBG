A. DAFTAR SCREENSHOT WAJIB (SESUAI PERMINTAAN)
==================================================

1) Screenshot Pemda: "Intelijen Rantai Pasok & GPS Tracker"
Cara Ambil:
- Buka: apps/pemda.html
- Klik menu samping: Tracker Progres Vendor
- Pilih salah satu vendor pada dropdown tracker
- Pastikan panel Google Maps tampil
- Pastikan tabel bawah terlihat (Kecepatan GPS, ETA, Risiko/Keterangan)
Caption Proposal:
Pemantauan Real-time Distribusi SPPG dengan integrasi Geospatial API dan Prediksi Risiko (XGBoost) untuk mengevaluasi Force Majeure.

2) Screenshot Pemda: "Auto-Rapor & E-BAST"
Cara Ambil:
- Buka: apps/pemda.html
- Klik menu: Auto-Rapor
- Klik tombol: Generate Auto-Rapor
- Tunggu preview dokumen putih (Canvas) muncul
- Screenshot area canvas + tombol Unduh PDF (jika terlihat)
Caption Proposal:
Generator Auto-Rapor Berbasis Gemini 1.5 (Strict RAG) untuk percepatan audit BPK dan pencairan BOSP.

3) Screenshot Guru: "YOLOv8 Edge Scanning"
Cara Ambil:
- Buka: apps/guru.html
- Login sebagai Guru
- Klik tab bawah: Scan (ikon kamera)
- Tekan tombol: Scan 1 Kedatangan Vendor
- Saat status menampilkan "Memindai..." / "YOLOv8 Offline", ambil screenshot
Caption Proposal:
Pemindaian Fleksibel (Scan Acak) menggunakan YOLOv8-Nano (Edge Computing) di gawai Guru tanpa bergantung pada koneksi internet.

4) Screenshot Wali Murid + Pemda: "Perlindungan KLB & Red Alert"
Cara Ambil:
- Buka: apps/wali.html (login sebagai Wali Murid)
- Masuk tab: Keluhan
- Kirim keluhan medis identik 3x dalam 6 jam (contoh teks: "Anak saya muntah setelah makan ayam")
  Catatan: sistem memakai cross-validation 3 keluhan identik/6 jam untuk trigger Red Alert.
- Lalu buka: apps/pemda.html
- Masuk menu: Red Alert
- Screenshot saat status "RED ALERT AKTIF" dan panel cross-validation terlihat
Caption Proposal:
Sistem Peringatan Dini (Early Warning System) dengan Cross-Validation Logic NLP untuk mencegah Kejadian Luar Biasa (KLB).

5) Screenshot Siswa: "Transparansi Menu Harian"
Cara Ambil:
- Buka: apps/murid.html (login sebagai Siswa)
- Buka tab: Home
- Pastikan foto/menu vendor terlihat
Caption Proposal:
Demokratisasi Pengawasan: Siswa dan Wali Murid mencocokkan secara mandiri foto prapengiriman dari SPPG.

6) Screenshot Super Admin (Opsional - Nilai Tambah Pitch Deck)
Cara Ambil:
- Buka: apps/super_admin.html
- Pastikan indikator Live Monitoring aktif
- Pastikan statistik monitoring terlihat

Command Center Terpusat: Ekosistem Multi-Aktor BERNAS MBG.
